class gianlp.models.text_representations.char_embedding_sequence.CharEmbeddingSequence(embedding_dimension: int = 256, sequence_maxlen: int = 80, min_freq_percentile: int = 5, random_state: int = 42)

Bases: TextRepresentation

Char embedding sequence input

Variables

  • _char_indexes – mapping from char to index

  • _keras_model – Keras model built from processing the text input

  • _embedding_dimension – the dimension of the embedding

  • _sequence_maxlen – the max length of an allowed sequence

  • _min_freq_percentile – the minimum frequency percentile to consider a char as known

  • _random_state – the random seed used for randomized operations

Parameters

  • embedding_dimension – The char embedding dimension

  • sequence_maxlen – The maximum allowed sequence length

  • min_freq_percentile – minimum percentile of the frequency for keeping a char. If a char has a frequency lower than this percentile it would be treated as unknown.

  • random_state – random seed
